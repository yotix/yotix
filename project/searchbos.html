Search Box

Seach box is a system to unify all the devices so we can easily access all the files on a single device without using other devices. All the documents can be viewed, videos and audios available in other device can be easily streamed over Wi Fi Connection with local LANs at home and with Internet when these devices are on different network. SearchBox has an additional features that separate it from others competitors as it allows sharing of these user based permissions for added security.
